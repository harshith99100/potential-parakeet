# Smart Energy Meter using ESP32

A real-time energy monitoring system that measures voltage, current, power, and energy consumption using embedded sensors and displays the data on an OLED screen.

## Features
- Voltage & current measurement
- Power calculation
- Energy estimation
- OLED display output
- Cloud-ready (IoT)

## Components Used
- ESP32
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- SSD1306 OLED

## Applications
- Smart homes
- Energy auditing
- IoT-based power monitoring
